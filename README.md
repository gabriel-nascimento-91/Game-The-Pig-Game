# 🐷 Game-The-Pig-Game 🐷
## Game rules:
- Roll the dice to accumulate points;
- If you roll a 1, it's the other player's turn;
- Choose hold to keep your score and pass the turn to the other player;
- First player to reach 50 wins!🏆🏆
